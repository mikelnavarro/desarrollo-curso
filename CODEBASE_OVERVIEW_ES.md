# Resumen del repositorio (ES)

Este repositorio es un espacio académico con muchas prácticas y proyectos independientes. No es una única aplicación.

## Estructura general
- Carpeta por unidades: `UD1-DWEC2526`, `UD5-DWEC2526`, `UD6-DWEC2526`, `UD7-DWEC2526`, etc.
- Otras prácticas/exámenes: `DESIGN`, `examen_recu`, `ProyectoTIENDA`, etc.
- El `README.md` indica que el repo sirve para ejercicios de HTML, CSS, JS, Java y PHP.

## Tipos de proyectos
- Ejercicios HTML/CSS simples con `index.html` y `styles.css`.
- Mini prácticas con JavaScript en carpetas de unidad (p. ej. `UD5-DWEC2526`).
- Proyectos con tooling (tienen `package.json`), como:
  - `examen_recu/angular`
  - `UD7-DWEC2526/examen_Ud07/web`
- Ejemplos de API/backend en `UD6-DWEC2526/api-coches`, etc.

## Carpeta observada recientemente
`UD5-DWEC2526/UT5_WEB_Practico` contiene:
- Un formulario de login (`index.html`) con su CSS.
- Una página `jugar.html` que carga scripts.
- Dos archivos añadidos para el Snake (solo UI por ahora):
  - `snake.html`
  - `snake.css`

## Cómo leer este repo
Piensa en él como una colección de ejercicios. Cada carpeta suele funcionar por separado y no hay un único punto de entrada.
